Web Inspector Options

Depth Control Settings

2

Set the default depth level for element inspection. Higher values will show more nested elements but may impact performance.

  • Level 1: Current element only
  • Level 2: Current element + direct children
  • Level 3: Current element + children + grandchildren
  • Level 4-5: Deeper nesting (use with caution on complex pages)

Measurement Mode Settings

Set the default measurement mode when the inspector is activated.

Keyboard Shortcuts:

  • Ctrl+Alt+A: Viewport Distance
  • Ctrl+Alt+S: Element Distance
  • Ctrl+Alt+Z: Margin Only
  • Ctrl+Alt+X: Padding Only
  • Ctrl+Alt+W: Child Elements
  • Ctrl+Alt+D: Size Only
  • Ctrl+Alt+E: Radius Only
