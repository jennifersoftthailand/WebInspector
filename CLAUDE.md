# Web Inspector

## Project Overview
A professional Chrome Extension for web designers, frontend developers, and UI engineers.
Provides real-time visual inspection and extraction of website design information.

## Core Features
- Element Inspection (width, height, margin, padding, border-radius, typography)
- Layout Analysis (distance measurement, element hierarchy, z-index)
- Visual Guides (highlight, custom border, colors)
- Design Export (save as HTML, preserve layout, offline reference)

## Technology Stack
- Chrome Extension (Manifest V3)
- Pure JavaScript (no framework dependency for performance)

## Competitive Position
Competes with: VisBug, CSS Peeper, Dimensions
Goal: unified solution combining measurement, inspection, extraction, offline preservation

## Monetization
Free: Inspection Features | Premium: AI Analysis, AI Export, Framework Conversion

## AI Instructions
Prioritize: Performance, Lightweight architecture, Browser compatibility
Never: Inject unnecessary scripts, Break host website behavior
